How to install c0lesgamingOS on your Computer.





To install c0lesgamingOS on your system, you need the official c0lesgamingOS ISO disc image.

Which you can download with the button bellow.




File SIze: 5.86GB





Method #1 (Windows 11 Installation Assistant):



After you have downloaded the disc image, Double-Click the file to "Mount" it to your Computer.

Now launch the setup.exe file licated in the "CD-ROM Drive".


The rest of the setup is just like installing WIndows 11 With the Windows Installation Assistant.




Method #2 (Bootable Media):



Warning: To do this method you will need an external program like rufus to continue with this method.



When you are using rufus to do this method of installation, You will need to set the File System to "NTFS" And set Partition Record To "GPT".

Then you want to load your iso (should be named "c0lesgamingOS v1.10.iso") onto your External Bootable Drive.

And then Reboot your system with the Bootable Media Inserted in your system.


Then select the bootable media from the Windows Boot Manager.

Then the typical Windows 11 Setup Wizard will start and you will be able to install c0lesgamingOS like you would with a regular install of Windows 11.




Method #3 (Manual Installation From Windows Recovery Command Prompt):



Warning: In order to execute this method, it is important that the ISO file is mounted to your system beforehand, as it cannot be mounted from the recovery environment.



Once you have the ISO file mounted on your keyboard hold the WIN+R Keys until you see the 'Run" dialog open on the Bottom-Left corner of your display,

Now type this text into the text box in the Run Dialog "ms-settings://", when the settings window opens up navigate to "Windows Update > Advanced options > Recovery".

And now under "Advanced Startup" click the "Restart" button, Then your computer will restart into the Windows Recovery Environment.


Now that you are in the Windows Recovery Environment (Windows RE) navigate to "Troubleshoot > Advanced Options > Command Prompt" and now you should be greeted with a familiar CMD WIndow.

From the command line you now have to manualy format the drive, copy the files, create registry entries, and reboot your system.
