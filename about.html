About c0lesgamingOS.





c0lesgamingOS was designed around bringing new life and performance into old, slow, Low-End machines.


c0lesgamingOS was made by gamers and optimized for a better gaming experience (P.S. We removed Edge :D)

In c0lesgamingOS we have also removed a lot of useless resource hogs including: Microsoft Edge, OneDrive, Microsoft Edge WebSockets and lots more!

And we have replaced them with better alternatives including the better web browser Firefox!


Preformance isn't the only thing we made better!

We also added some UI tweaks to the Windows 11 UX using the Registry and Rectify11!


c0lesgamingOS was built around Microsoft Windows 11 using the NTLite program.
